#1</br>
Q: What is data?</br>
A: Data is information that can be computed, moved or processed by the computer.</br>

#2</br>
Q: What is a value?</br>
A: Value is something that has a meaning and is represented as data.</br>

#3</br>
Q: What is an object?</br>
A: Object is a region of storage that can hold something.</br>

#4</br>
Q: What is a variable?</br>
A: Variable is an object that has a name</br>

#5</br>
Q: What is an identifier?</br>
A: Identifier is the name of something. (ex, name of a variable)</br>

#6</br>
Q: What is a data type used for?</br>
A: To define and identify variable/object holds what data</br>

#7</br>
Q: What is an integer?</br>
A: Integer is a 32 bit number data type (int)</br>
