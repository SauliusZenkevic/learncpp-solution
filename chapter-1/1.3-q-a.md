#1
Q: What is data?
A: Data is information that can be computed, moved or processed by the computer.

#2
Q: What is a value?
A: Value is something that has a meaning and is represented as data.

#3
Q: What is an object?
A: Object is a region of storage that can hold something.

#4
Q: What is a variable?
A: Variable is an object that has a name

#5
Q: What is an identifier?
A: Identifier is the name of something. (ex, name of a variable)

#6
Q: What is a data type used for?
A: To define and identify variable/object holds what data

#7
Q: What is an integer?
A: Integer is a 32 bit number data type (int)
