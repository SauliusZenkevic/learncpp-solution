Q: What is the difference between initialization and assignment?
A: Initialization is a variable initialization, when initial data is being assigned to the variable and assignment is when the variable gets assigned to a value or different data.

#3
Q:What are default-initialization and value-initialization? What is the behavior of each? Which should you prefer?
A: default initializaion is when the variable gets initialized with initial data. Value initialization is when the variable gets initialized with some sort of value or data. Value initialization should be prefeared, because default initialization assigned initial data or value is taken from uncleaned stack. For example, initial data can be random numbers or god knows what.
