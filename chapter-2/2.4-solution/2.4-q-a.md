#1</br>
Q: What’s wrong with this program fragment?</br>
```cpp
#include <iostream>

void multiply(int x, int y)
{
    return x * y;
}

int main()
{
    std::cout << multiply(4, 5) << '\n';

    return 0;
}
```
A: Function multiply cannot return anything, because its return type is void. This progam will not compile.</br>

#3</br>
Q: What value does the following program print?</br>
```cpp
#include <iostream>

int add(int x, int y, int z)
{
    return x + y + z;
}

int multiply(int x, int y)
{
    return x * y;
}

int main()
{
    std::cout << multiply(add(1, 2, 3), 4) << '\n';

    return 0;
}
```
A: 24</br>
