Q: Inspect the following programs and state what they output, or whether they will not compile.</br>
a)</br>
```cpp
#include <iostream>

void printA()
{
    std::cout << "A\n";
}

void printB()
{
    std::cout << "B\n";
}

int main()
{
    printA();
    printB();

    return 0;
}
```
b) </br>
```cpp
#include <iostream>

void printA()
{
    std::cout << "A\n";
}

int main()
{
    std::cout << printA() << '\n';

    return 0;
}
```
A:</br>
a) This program is going to output:</br>
A</br>
N</br>
b) The code is not going to compile, because printA function does not return any value that could be streamed to the console.</br>
